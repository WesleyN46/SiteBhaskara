Passo a Passo para Calcular a Fórmula de Bhaskara:

1. Identificar os coeficientes: Na equação ax² + bx + c = 0, identifique os valores de a, b e c.

2. Calcular o Delta (Δ): Use a fórmula: Δ = b² - 4ac.

3. Analisar o Delta: Se Δ < 0, não existem raízes reais. Se Δ = 0, existe uma raiz real. Se Δ > 0, existem duas raízes reais.

4. Calcular as raízes: Se Δ > 0, as raízes podem ser calculadas por:
Raiz 1 = (-b + √Δ) / 2a
Raiz 2 = (-b - √Δ) / 2a
